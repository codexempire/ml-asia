## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository, implements a user registration and retreival route.

## Tools
- Mongoose and MongoDB - ORM connector and Database
- Typescript
- Jsonwebtoken - sign and verify token
- bcrypt - hashing of passwords

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
